# Material UI

Material-ui is awesome framework to use for web/app, it supports JS/TS both.

```git
// with npm
npm install @material-ui/core

//with yarn
yarn add @material-ui/core
```

# React Bootstrap

Bootstrap is one of my best front-end frameworks

```git
// with npm
npm install react-bootstrap bootstrap@4.6.0

//with yarn
yarn add react-bootstrap bootstrap@4.6.0
```

# Semantic-ui React

Semantic is an easy and friendly framework to use for projects, it provides a lot of components.

```git
// with npm
npm install semantic-ui-react semantic-ui-css

//with yarn
yarn add semantic-ui-react semantic-ui-css
```

# Onsen UI

If you're creating a cross-platform application you'll be interested in this

```git
// with npm
npm install onsenui react-onsenui

//with yarn
yarn add onsenui react-onsenui
```

# Evergreen

Coolest and simplest front end framework, you'll love once you look at it!

```git
// with npm
npm install evergreen-ui

//with yarn
yarn add evergreen-ui
```
